# [`spTools`](https://github.com/cwendorf/spTools/)

## Reference

This section is the reference for `spTools`. Each page documents one exported function, including usage, arguments, return values, and examples.

### Vectorized Versions

- [ci.mean.vec()](./ci.mean.vec.md): Confidence intervals for a set of means with vector input.
- [ci.mean.ps.vec()](./ci.mean.ps.vec.md): Wrapper for `ci.mean.ps` with vector input.
- [ci.mean2.vec()](./ci.mean2.vec.md): Wrapper for `ci.mean2` with vector input.
- [ci.stdmean.ps.vec()](./ci.stdmean.ps.vec.md): Wrapper for `ci.stdmean.ps` with vector input.
- [ci.stdmean2.vec()](./ci.stdmean2.vec.md): Wrapper for `ci.stdmean2` with vector input.

### Additional Functions

- [ci.lc.mean.ws()](./ci.lc.mean.ws.md): Confidence interval for a linear contrast of means (within-subjects).
- [ci.mean2.compare()](./ci.mean2.compare.md): Compare two means with group and difference confidence intervals.

### Output Formatting

- [ci.add.test()](./ci.add.test.md): Add test statistic, df, and p-value to CI output.
- [ci.tukey.reformat()](./ci.tukey.reformat.md): Clean and reformat Tukey CI output.
